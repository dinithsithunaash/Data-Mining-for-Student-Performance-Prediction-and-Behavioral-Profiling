# Student Performance Prediction Notebook

This notebook is my end-to-end data mining workflow for predicting student performance and understanding learner behavior using the OULAD dataset.

It includes:
- data loading and preprocessing
- feature engineering
- supervised models (Logistic Regression, Random Forest, XGBoost)
- hyperparameter tuning
- unsupervised clustering (K-Means + PCA)
- visual analysis and result interpretation

## Why this work matters
The goal is simple: identify at-risk students early and support them in time.
